Layout Features:

  • Variable Aspect Ratios: Mixes 16:9, 1:1, and 2:1 cards to break the repetition of a standard grid.
  • Visual Hierarchy: Uses size to signal importance (e.g., currently watching vs. recommendations).
  • Expressive Shapes: Leverages the `extraLarge` (24dp+) corner radii from the Material 3 Expressive theme.
  • Color Grouping: Uses container colors to subconsciously categorize content types (Music vs. Movies).
