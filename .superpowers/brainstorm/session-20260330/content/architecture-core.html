// Orbit MVI Base ViewModel
abstract class BaseMviViewModel<S : Any, SE : Any> : ViewModel(), ContainerHost<S, SE> {
  abstract val initialState: S
  override val container by lazy { container<S, SE>(initialState) }

  // Helper for intent-based actions
  protected fun updateState(reducer: S.() -> S) = intent { reduce { state.reducer() } }
}

Key Components

  • Orbit MVI: Chosen for its balance of strictness and boiler-plate reduction. It handles state reduction and side effects (toasts, navigation) elegantly.
  • State-In-Container: ViewModels will no longer hold individual MutableStateFlows. All UI state resides in a single, immutable UiState object.
  • SideEffects: One-time events (like showing a Snackbar or navigating) are modeled as SideEffects, preventing "double-fire" issues on configuration changes.
  • Compose Integration: Uses collectAsStateWithLifecycle() for state and collectSideEffect() for events.
